# PHPServer

Simple PHP server for development.

## Installation

Add this line to your application's Gemfile:

    gem 'php-server'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install php-server

## Usage

    $ phpserver

### CakePHP

    $ phpserver cakephp

### FuelPHP

    $ phpserver fuelphp

## Required

* php-cgi
